Diagramm-Notizen
================

Dieser Ordner ist fuer Diagrammquellen gedacht, zum Beispiel .excalidraw-Dateien.

Empfehlung
----------

- Excalidraw-Dateien hier als *.excalidraw.json ablegen
- Beim lokalen dev/build werden sie automatisch nach docs/assets/images/*.svg exportiert
- In Markdown immer auf die exportierte SVG-Datei verlinken

So bleiben die editierbaren Quellen und die eingebundenen Bilder sauber getrennt.
